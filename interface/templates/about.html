About Our Project

AI-powered sugarcane disease and pest detection using YOLOv8 deep learning models.

Project Information

The AI Crop Disease Detector is an advanced agricultural intelligence platform that leverages state-of-the-art YOLOv8 computer vision models to identify diseases and pest infestations in sugarcane crops in real-time.

Our system provides instant diagnostic reports with treatment recommendations, helping farmers take timely action to protect their crops and maximize yield.

Technology Stack

FastAPI YOLOv8 PyTorch OpenCV SQLite TailwindCSS JavaScript

Key Features

  • ✓ Real-time disease & pest detection with bounding box overlays
  • ✓ Live webcam analysis with instant detection feedback
  • ✓ Grad-CAM spatial activation heatmaps
  • ✓ Dynamic treatment & prevention recommendations
  • ✓ Downloadable PDF diagnostic reports
  • ✓ Scan history dashboard with SQLite persistence
